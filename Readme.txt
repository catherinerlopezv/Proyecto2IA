# 📷 Detector de Emociones en Tiempo Real

Este proyecto implementa un sistema de inferencia por cámara que detecta emociones faciales en tiempo real utilizando un modelo de deep learning entrenado con Keras y TensorFlow.

## 🚀 Características

- Captura imágenes desde la cámara web.
- Predice la emoción dominante entre 7 clases: `Angry`, `Disgust`, `Fear`, `Happy`, `Neutral`, `Sad`, `Surprise`.
- Muestra la predicción con porcentaje de confianza.
- Visualiza barras de predicción para todas las emociones.
- Incluye un contador acumulado por emoción para múltiples capturas.
- Interfaz visual moderna y compacta, optimizada para no salirse de pantalla.
